Handloom Park | Luxury Saree E-commerce
A sleek, responsive, and luxury-focused web application showcasing traditional Sri Lankan handloom and textile collections. This project was developed as part of a digital transformation initiative for Handloom Park, focusing on high-end presentation and seamless customer interaction via WhatsApp integration.

🌟 Features
Luxury Aesthetic: Minimalist dark-themed UI designed with a focus on high-quality fabric photography.

Collection Management: Dedicated sections for Cotton, Pooni, and Screen Print collections.

Fully Responsive: Optimized for mobile, tablet, and desktop browsing using Bootstrap 5.

Direct-to-WhatsApp Ordering: Integrated CTA buttons for instant customer inquiries and sales.

Performance Focused: Built with clean HTML5 and CSS3 for fast loading times.

🛠️ Technology Stack
HTML5 - Structured content.

CSS3 - Custom styling with Google Fonts integration.

Bootstrap 5 - Responsive grid system and components.

GitHub Pages - Production hosting.

📂 Project Structure
index.html — The main landing page showcasing the brand and collection highlights.

cotton.html — Catalog for the 17-piece Cotton collection.

screenprint.html — Catalog for the 56-piece Screen Print collection.

images/ — Organized directory for high-resolution product photography.
